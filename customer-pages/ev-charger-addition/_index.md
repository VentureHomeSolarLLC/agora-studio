# Can I Add an EV Charger?

**Yes** — and you don't need to be a Venture Home solar customer to get one.

We install EV chargers for any homeowner in our service area.

---

## What We Install

We install **Level 2 home chargers** — the right solution for overnight home charging.

**Level 2 specs:**
- Adds **20-30 miles of range per hour** of charging
- Most drivers wake up to a **full battery every morning**

---

## Charger Options

We can install **any charger brand**, but we recommend two:

### Tesla Universal Wall Connector

- ✓ Works with **any EV** (not just Teslas)
- ✓ Sleek design
- ✓ Smart charging features
- ✓ Integrates with Tesla solar and Powerwall

**Best for:** Tesla solar/Powerwall owners, or any EV driver wanting premium features

### Enphase EV Charger

- ✓ Integrates natively with **Enphase solar and battery systems**
- ✓ **Solar-prioritized charging** — maximizes clean energy use
- ✓ Minimizes grid draw

**Best for:** Enphase solar/battery owners wanting seamless integration

### Recommendation

If you already have an Enphase or Tesla solar system, **the matching charger gives you the best integration and monitoring experience**.

If you have a different system or **no solar at all**, either charger works well.

---

## Panel Upgrades

If your electrical panel **doesn't have capacity** for a Level 2 charger:

- ✅ We handle the **panel upgrade** as part of the project
- ✅ **You don't need a separate electrician**
